# Projeto-Jogo-Educacional
Um pequeno quiz para definir o quão Brusquense você é.
Aqui você irá responder 15 questões sobre Brusque de variados níveis de dificuldade, e no final iremos ver o quão Brusquense você, tendo perguntas sobre a história, cultura e a atualidade de Brusque.

Feito por: Bruno Avi, Higor Bueno Ferreira, Matheus Becker Bittelbrunn e Izack Kaiki Kohler
